# GEARCALC-PRO-DEMO-V2

Prototype demo for GEARCALC-PRO with:
- Frontend (HTML/CSS/JS)
- Backend (FastAPI)
- MATLAB wrapper (optional mode)
- Python placeholder model (NN demo)

## Modes
- **Python-only demo mode** (runs immediately)
- **MATLAB->Python pipeline mode** (set `USE_MATLAB=1` and ensure MATLAB is installed)

## Local run (quick)
1. Start backend from `backend/`
2. Start frontend static server from `frontend/`
3. Open browser and run the demo

See code comments and setup instructions in chat notes / project docs.
